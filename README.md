## **Project Description** 
This project is a **compiler written in TypeScript** for the CMPT432 Course  
**Goal:** Design, implement, and test a compiler from scratch

---

### **Development Environment Setup**
Ensure you have installed: 
- **[Node.js]** 
- **Git** 
- **A Code Editor** 

---

#### **Run & Verify Installations** 
```sh 
npm install # (TypeScript, ts-node, nodemon, ESLint)
node -v # Check Node.js version
npm -v  # Check npm version 
git --version # Check Git version 
```

---
##### **Running the Project** 
1. Development Mode 
```sh 
npm run dev # Runs the project using ts-node 
```
2. Building the Project 
```sh 
npm run build # Compiled files will be in the dist/ folder 
```
3. Running Compiled Code 
```sh 
npm start # Runs dist/index.js using Node.js
```

---

###### **Testing & Debugging** 
Running ESLint (Code Linter)
Check for TypeScript Errors:
```sh
npm run lint
```
